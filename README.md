# TypeTrack – Typing Speed Web App
TypeTrack – Full Stack Typing Speed Web Application

Live Demo: https://typetrack.vercel.app 

## Overview

TypeTrack is a full-stack typing speed test platform that allows users to measure their typing performance in real-time.  

The app measures:
Words Per Minute (WPM)
Characters Per Minute (CPM)
Accuracy percentage
All results are stored securely in MongoDB so users can track their progress over time.

##  Features

-  JWT-based User Authentication (Login/Register)
-  Real-time 60-second typing test
-  WPM, CPM, and Accuracy calculation
-  100+ randomized typing paragraphs
-  Light/Dark Mode toggle
-  Dashboard with performance history
-  Fully deployed (Frontend + Backend)
